# DOC DOCS - NEXTJS DOCUMENTATION

doc_docs is for NextJS projects, he use the file routing method of NextJS to create a custom documentation.

### HOW TO INSTALL

```
npm i doc_docs
```

### RULES

> #### `src:`
>
> your project needs a src folder with the "pages" of next js inside.
